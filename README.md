## Installation

1. From vscode

   - Launch VS Code Quick Open (Ctrl+P)
   - Paste `ext install picomet.alpinejs`
   - Press enter

2. From cli

   - Open terminal
   - Enter : `code --install-extension picomet.alpinejs`
